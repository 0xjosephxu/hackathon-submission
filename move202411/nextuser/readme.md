## project
- 项目名称:存钱有奖
> 描述: 
1. 收集n个用户存储的货币，
2. 将货币质押或提供流动性给一个靠谱的第三方dex 或质押方。 目前质押给validator 节点
3. 每隔一段时间，提取存款的利率的50% 根据随机抽奖，作为奖品发给中奖的m个用户， m 小于 n，
剩余的存款和利息，继续质押给第三方。
4. 用户的存款本金不会损失
5. 运营方初期可以投入奖金鼓励储蓄。
