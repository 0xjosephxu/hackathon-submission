# Project
- 项目名称：Crab
  描述：
     Crab项自旨在通过社区驱动的方式，建立一个安全、透明的加密货币投资环境。它利用投票标记统让用户参与到诈骗币 （Scam Coin）的识别过程中，从而增强整个区块链生态系统的安全性与健康度。
